A Short Showcase of My Anime/Manga Reader Web Design Side Project

I've been a fan of anime and manga for as long as I can remember. I love the stories, the characters, and the art. But I've always found it difficult to find a good website to read manga online. Many of the websites I've tried are poorly designed, have a lot of ads, or are missing the latest chapters.

That's why I decided to create my own website. I wanted to create a website that was easy to use, had no ads, and always had the latest chapters of my favorite manga.

I have a lot of plans for the future of the website. I want to add more features, such as a manga recommendation system and a feature that allows users to download manga chapters. I also want to translate the website into different languages.

Site Link: https://kotokahatsushima.github.io/MangaAnime/MangaAnimeSite/

GitHub Link: https://github.com/Kotokahatsushima/MangaAnime
